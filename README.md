# 🎨 Vedhavathi Chennupati - Animated Portfolio

A stunning, fully-animated single-page portfolio website showcasing my projects, skills, and experience as a CS Engineering student.

## ✨ Features

- 🎯 **Custom Cursor Animation** - Glowing dot with trailing ring effect
- ✨ **Live Particle Network** - Floating particles with connecting lines
- 📊 **Animated Skill Bars** - Bars fill on scroll with shimmer effect
- 🎬 **Smooth Scroll Animations** - All elements animate in as you scroll
- 💼 **Interactive Timeline** - Experience and involvement timeline
- 📱 **Fully Responsive** - Beautiful on desktop, tablet, and mobile
- ⚡ **Pure HTML/CSS/JS** - No frameworks, pure vanilla JavaScript
- 🎨 **Modern Design** - Dark theme with vibrant accent colors
- 🚀 **Vercel Deployed** - Fast, automatic deployments on push

## 🎨 Design Highlights

- **Color Palette:** Deep indigo, hot pink (#ec4899), cyan (#06b6d4), amber (#f59e0b)
- **Typography:** Syne (headings) + DM Sans (body)
- **Animations:** 15+ unique CSS animations
- **Loading Screen:** 2-second animated gradient progress bar
- **Particles:** 50 floating particles with intelligent connections

## 🚀 Quick Start

### Local Development

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/portfolio.git

# Navigate to the folder
cd portfolio

# Open in browser
open index.html
# or on Windows:
start index.html
```

### Deploy to Vercel

1. Push code to GitHub
2. Go to [Vercel.com](https://vercel.com)
3. Import your GitHub repository
4. Click Deploy
5. Your site is live! 🎉

Live Demo: https://portfolio-YOUR_USERNAME.vercel.app

## 📝 Customization

To customize your portfolio:

1. **Change Name/Info:**
   - Edit "Vedhavathi Chennupati" with your name
   - Update email, phone, location

2. **Update Skills:**
   - Modify skill names and percentages in Skills section

3. **Add Projects:**
   - Duplicate project cards and update details

4. **Change Colors:**
   - Update CSS variables in `:root` selector

5. **Add Social Links:**
   - Add your LinkedIn, GitHub links to navbar

3. Add your domain
4. Update DNS records according to Vercel's instructions
